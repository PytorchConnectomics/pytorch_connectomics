# MMAAPP

**GitHub:** https://github.com/seung-lab/MMAAPP
**Language:** C++ | **Stars:** 2

Mostly Mean Affinity Agglomeration and Post-Processing. C++ implementation of affinity-based agglomeration for neuron segmentation.

## Key Features
- Mean affinity agglomeration algorithm
- Post-processing pipeline for segment merging
- C++ implementation for performance

## Relevance to Connectomics
Implements agglomerative clustering on affinity graphs to produce neuron instance segmentations from boundary predictions, a core step in the EM segmentation pipeline.
