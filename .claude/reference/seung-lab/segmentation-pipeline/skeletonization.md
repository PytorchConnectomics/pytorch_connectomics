# skeletonization

**GitHub:** https://github.com/seung-lab/skeletonization
**Language:** Python | **Stars:** 9

Reconstructed object skeletonization pipeline. **Deprecated** -- replaced by [kimimaro](https://github.com/seung-lab/kimimaro).

## Key Features
- Point cloud generation from segmentation volumes
- TEASAR-based skeletonization
- Post-processing of skeleton graphs

## Relevance to Connectomics
Early skeletonization tool for neuron reconstructions; superseded by kimimaro which is used as a dependency in PyTC.
