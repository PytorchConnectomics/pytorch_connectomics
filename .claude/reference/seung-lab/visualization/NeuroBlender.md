# NeuroBlender

**GitHub:** https://github.com/seung-lab/NeuroBlender
**Language:** Jupyter Notebook | **Stars:** 8

Tutorial and source code for importing neuron meshes, segmentation images, and EM images into Blender for 3D visualization and animation, using CloudVolume and MeshParty.

## Key Features
- Import neuron meshes and EM images into Blender
- Correct scaling of images and meshes
- Neuron coloring and material setup
- Rendering neuron animations
- Uses CloudVolume, MeshParty, and Open3D

## Relevance to Connectomics
Creates publication-quality 3D renderings and animations of reconstructed neurons from EM connectomics datasets.
