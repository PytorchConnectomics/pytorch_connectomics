# omni.play

**GitHub:** https://github.com/seung-lab/omni.play
**Language:** C++ | **Stars:** 4

2D/3D neuron reconstruction tool (OMNI). A desktop application for proofreading and visualizing EM segmentation volumes, with dependencies on Boost, Thrift, zlib, and libjpeg-turbo.

## Key Features
- Interactive 2D and 3D visualization of EM segmentation data
- Manual proofreading and correction of neuron reconstructions
- Volume-based segmentation viewing

## Relevance to Connectomics
Desktop proofreading tool for manual correction of automated EM neuron segmentations.
