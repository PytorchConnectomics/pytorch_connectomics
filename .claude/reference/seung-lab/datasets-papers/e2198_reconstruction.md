# e2198_reconstruction

**GitHub:** https://github.com/seung-lab/e2198_reconstruction
**Language:** Matlab | **Stars:** 0

Data and code for visualizing connectomics reconstructions of the e2198 retinal dataset. Contains cell reconstruction processing pipeline from Omni segmentation volumes to downsampled/warped indexed images, cell_info metadata structures, and stratification profile computation.

## Key Features
- Cell reconstruction pipeline from Omni to HDF5
- Conformal warping of cell volumes for retinal flattening
- cell_info metadata structure (cell ID, class, type, soma, stratification)
- Stratification profile computation and visualization
- Soma removal for dendritic-only analysis

## Relevance to Connectomics
Data processing and visualization pipeline for the e2198 retinal EM reconstruction dataset.
