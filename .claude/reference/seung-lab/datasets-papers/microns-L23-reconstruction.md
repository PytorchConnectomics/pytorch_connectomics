# microns-L23-reconstruction

**GitHub:** https://github.com/seung-lab/microns-L23-reconstruction
**Language:** None | **Stars:** 2

A record of the reconstruction code used for the L2/3 IARPA MICrONS (Machine Intelligence from Cortical Networks) reconstruction project.

## Relevance to Connectomics
Documents the reconstruction pipeline for the MICrONS cortical layer 2/3 dataset, one of the largest mammalian connectomics reconstructions.
