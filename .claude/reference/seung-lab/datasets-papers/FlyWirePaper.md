# FlyWirePaper

**GitHub:** https://github.com/seung-lab/FlyWirePaper
**Language:** Jupyter Notebook | **Stars:** 3

Source code and data to reproduce analyses from the FlyWire paper. Contains Jupyter notebooks and datasets for recreating figures from the whole-brain Drosophila connectome publication.

## Key Features
- Jupyter notebooks for each figure in the FlyWire paper
- Bundled data files for reproducibility
- Installable plotting module

## Usage
```bash
cd FlyWirePaper
pip install . --upgrade
# Run notebooks in /flywirepaper with "Run all"
```

## Citation
Dorkenwald et al., "FlyWire: Online community for whole-brain connectomics," bioRxiv, 2020.

## Relevance to Connectomics
Documents the analysis pipeline for the FlyWire whole-brain Drosophila connectome, one of the landmark connectomics reconstruction projects.
