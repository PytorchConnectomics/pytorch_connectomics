# e2198_Ca_imaging

**GitHub:** https://github.com/seung-lab/e2198_Ca_imaging
**Language:** Matlab | **Stars:** 1

Cell clustering and calcium imaging analysis of the e2198 retinal dataset (Briggman et al. 2011). Contains code and data for calcium imaging analysis, cell correspondence mapping between calcium imaging ROIs and EM reconstructions, and cell clustering based on anatomy and physiology.

## Key Features
- Calcium imaging data (DeltaF/F traces, direction/orientation selectivity)
- Cell mapping between EM cell IDs and calcium ROI indices
- Stimulus response parameter fitting
- Soma coordinate data in warped coordinates

## Relevance to Connectomics
Links functional calcium imaging data with EM structural reconstructions for retinal ganglion cell classification.
