# e2198-gc-analysis

**GitHub:** https://github.com/seung-lab/e2198-gc-analysis
**Language:** MATLAB | **Stars:** 8

Connectomics analysis of retinal ganglion cells traced in the e2198 EM dataset. Contains code and data for cell type classification via hierarchical clustering, dendritic arbor analysis, stratification profiles, calcium imaging correlation, and soma measurements.

## Key Features
- Hierarchical clustering of ganglion cells by anatomy and physiology
- Dendritic arbor skeletonization and asymmetry analysis
- Stratification profile computation in the IPL
- Calcium imaging direction/orientation selectivity integration
- Soma size measurement via ConvNet segmentation
- 3D mesh data for all reconstructed ganglion cells

## Relevance to Connectomics
Comprehensive analysis codebase for retinal ganglion cell classification using combined EM reconstruction and functional imaging data.
