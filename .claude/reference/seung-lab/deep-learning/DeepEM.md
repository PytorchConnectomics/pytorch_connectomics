# DeepEM

**GitHub:** https://github.com/seung-lab/DeepEM
**Language:** Python | **Stars:** 16

Deep learning framework for EM connectomics, achieving superhuman accuracy on the SNEMI3D challenge (Lee et al. 2017). Used in the large-scale cortical reconstruction by Dorkenwald et al. 2019.

## Key Features
- 3D convolutional network training for EM segmentation
- Boundary detection / affinity prediction
- Training infrastructure for large-scale EM datasets

## Relevance to Connectomics
One of the foundational deep learning frameworks for EM neuron segmentation from the Seung Lab, a predecessor to modern approaches used in PyTC.
