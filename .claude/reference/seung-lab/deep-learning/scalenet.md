# scalenet

**GitHub:** https://github.com/seung-lab/scalenet
**Language:** None | **Stars:** 1

A universal PyTorch module for implementing multiscale architectures (UNet, spatial pyramid, SpyNet, res-UNet). Provides building blocks for constructing multi-resolution neural network architectures.

## Key Features
- Universal multi-scale architecture building blocks
- Supports UNet, spatial pyramid, SpyNet, and res-UNet patterns
- PyTorch-native module design

## Relevance to Connectomics
Provides reusable multi-scale architecture components used by MetroEM and other Seung Lab alignment/segmentation models.
