# devoem

**GitHub:** https://github.com/seung-lab/devoem
**Language:** None | **Stars:** 0

Dense Voxel Embeddings for EM Connectomics. Research code for learning and segmenting dense voxel embeddings for 3D neuron reconstruction (Lee et al., IEEE TMI 2021).

## Key Features
- Dense voxel embedding learning for 3D EM volumes
- Instance segmentation via embedding clustering
- Companion code for Lee et al. 2021 paper

## Citation
Lee et al., "Learning and Segmenting Dense Voxel Embeddings for 3D Neuron Reconstruction," IEEE TMI, 2021.

## Relevance to Connectomics
Proposes an embedding-based approach to neuron instance segmentation from EM data, an alternative to affinity-based methods used in PyTC.
