# metric_segmentation

**GitHub:** https://github.com/seung-lab/metric_segmentation
**Language:** Jupyter Notebook | **Stars:** 3

Segmenting EM images of neurons using deep metric learning. Learns embedding spaces where pixels belonging to the same neuron are close together.

## Key Features
- Deep metric learning approach to EM segmentation
- Pixel embedding for instance segmentation
- Jupyter notebook-based workflow

## Relevance to Connectomics
Explores metric learning as an alternative to affinity-based approaches for neuron instance segmentation in EM volumes.
