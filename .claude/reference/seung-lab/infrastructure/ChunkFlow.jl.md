# ChunkFlow.jl

**GitHub:** https://github.com/seung-lab/ChunkFlow.jl
**Language:** Julia | **Stars:** 1 | **Archived**

Distributed pipeline for chunk-based processing inside and beyond the cloud. Deprecated in favor of the Python version (chunkflow). Defined computation as a node graph, with task scheduling via AWS SQS.

## Key Features
- JSON-defined task pipelines (node-based computation graph)
- AWS SQS-based task distribution
- Multi-process execution

## Relevance to Connectomics
Early Julia framework for distributed chunk-based EM volume processing; superseded by Python chunkflow.
