# CAVEpipelines

**GitHub:** https://github.com/seung-lab/CAVEpipelines
**Language:** HCL | **Stars:** 5

Instructions and infrastructure-as-code for creating ChunkedGraph instances, initial meshing, and L2 cache for the CAVE (Connectome Annotation Versioning Engine) system.

## Key Features
- ChunkedGraph creation pipeline
- Initial mesh generation from segmentation
- Level 2 cache setup
- Terraform/HCL infrastructure definitions
