# cell-picker

**GitHub:** https://github.com/seung-lab/cell-picker
**Language:** Julia | **Stars:** 1

Seed new EyeWire cells by given Neuroglancer or EyeWire coordinates. Used to spawn tracing tasks for the EyeWire citizen science platform.

## Key Features
- Accepts coordinates in Neuroglancer (NG) or EyeWire (EW) coordinate systems
- Configurable cell properties (name, difficulty, display, duplicate detection)
- Direct MySQL integration with EyeWire database

## Relevance to Connectomics
Internal tooling for spawning neuron tracing tasks in the EyeWire crowd-sourced connectomics reconstruction project.
