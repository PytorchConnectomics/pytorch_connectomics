# ng-extend

**GitHub:** https://github.com/seung-lab/ng-extend
**Language:** Vue | **Stars:** 5

Extended Neuroglancer viewer built as a dependent project using git submodules. Provides custom extensions on top of Google's Neuroglancer 3D viewer.

## Key Features
- Custom extensions to Neuroglancer
- Built with Vue.js frontend
- Uses git submodule for Neuroglancer core

## Relevance to Connectomics
Provides enhanced 3D visualization of EM image volumes and segmentation data beyond stock Neuroglancer.
