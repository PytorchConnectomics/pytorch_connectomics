# gallery

**GitHub:** https://github.com/seung-lab/gallery
**Language:** Stylus (Node.js) | **Stars:** 3

The Museum -- a web repository for visualizing connectomic reconstructions of neurons from Seung Lab. Displays 3D meshes, classification data, stratification profiles, and calcium response curves for retinal ganglion cells from the e2198 dataset.

## Key Features
- 3D neuron mesh visualization in browser
- Cell type classification display
- Stratification profile and calcium imaging data integration
- MongoDB-backed with Node.js server

## Relevance to Connectomics
Web-based visualization platform for browsing and sharing reconstructed neuron morphologies from EM datasets.
