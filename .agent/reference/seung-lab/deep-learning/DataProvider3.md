# DataProvider3

**GitHub:** https://github.com/seung-lab/DataProvider3
**Language:** Jupyter Notebook | **Stars:** 3

Platform-independent volumetric data provider for training 3D convolutional networks. Developed for the superhuman SNEMI3D connectomics challenge (Lee et al. 2017).

## Key Features
- Deep learning framework-agnostic volumetric data loading
- 3D patch sampling for convolutional network training
- Designed for EM connectomics data

## Relevance to Connectomics
Provides volumetric data loading for training 3D segmentation networks on EM data, a predecessor to modern data loading in PyTC.
