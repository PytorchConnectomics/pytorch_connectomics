# modelhouse

**GitHub:** https://github.com/seung-lab/modelhouse
**Language:** Python | **Stars:** 0

Distribute and load ML models from cloud storage backends (GCS, S3, local). Provides model caching for reduced loading time on cloud workers.

## Key Features
- Store/load models from GCS, S3, or local filesystem
- Model caching for reduced load times
- Designed for distributing models to cloud workers

## Installation
```bash
pip install modelhouse
```

## Relevance to Connectomics
Manages distribution of trained segmentation models to inference workers in cloud-based EM processing pipelines.
