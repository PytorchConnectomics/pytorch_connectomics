# ZNNi

**GitHub:** https://github.com/seung-lab/ZNNi
**Language:** C++ | **Stars:** 1

ZNNi -- Maximizing 3D ConvNet Inference Throughput Using Multi-Core, Many-Core CPUs and GPUs. Optimized inference engine for 3D convolutional networks, supporting Intel TBB, CUDA 7.5, and cuDNN.

## Key Features
- Hybrid CPU/GPU 3D ConvNet inference
- Intel compiler + TBB for multi-core CPU parallelism
- CUDA/cuDNN GPU acceleration
- Optimized for dense sliding-window output

## Relevance to Connectomics
High-throughput inference engine for running trained 3D ConvNets on large EM volumes; companion to ZNN training framework.
