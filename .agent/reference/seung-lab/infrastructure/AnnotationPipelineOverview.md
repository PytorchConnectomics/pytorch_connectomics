# AnnotationPipelineOverview

**GitHub:** https://github.com/seung-lab/AnnotationPipelineOverview
**Language:** None | **Stars:** 2

Documentation repository providing an overview of the CAVE (Connectome Annotation Versioning Engine) system architecture, linking together all services, client libraries, and deployment repos.

## Key Components Documented
- Services: NeuroglancerJsonServer, PyChunkedGraph, AnnotationEngine, MaterializationEngine
- Client libraries: CAVEClient, MeshParty, PCG_skel
- Viewer: Neuroglancer
- Deployment: CAVEDeployment

## Relevance to Connectomics
Documents the full CAVE annotation and proofreading infrastructure used for large-scale connectomics reconstruction projects like FlyWire.
