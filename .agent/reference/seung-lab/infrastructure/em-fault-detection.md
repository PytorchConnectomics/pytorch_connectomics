# em-fault-detection

**GitHub:** https://github.com/seung-lab/em-fault-detection
**Language:** Python | **Stars:** 0

Online image analysis for detecting high-consequence errors during high-throughput electron microscopy (e.g., tears in Kapton tape-fed TEM imaging).

## Key Features
- Real-time fault detection during automated EM imaging
- Three severity levels: HALT, WARNING, OK
- Analysis of low and high magnification images
- Designed for tape-based TEM workflows

## Relevance to Connectomics
Quality control during EM data acquisition to prevent imaging failures that would disrupt serial section datasets.
