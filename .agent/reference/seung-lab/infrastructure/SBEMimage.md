# SBEMimage

**GitHub:** https://github.com/seung-lab/SBEMimage
**Language:** N/A (Python/PyQt) | **Stars:** 0

Fork of SBEMimage, an open-source acquisition software for serial block-face electron microscopy (SBEM). Controls the microscope during automated volumetric EM data collection.

## Key Features
- Automated SBEM acquisition control
- Built with Python and PyQt
- Supports serial block-face imaging workflows

## Relevance to Connectomics
Controls the data acquisition stage of the EM connectomics pipeline, producing the raw image volumes that are later segmented.
