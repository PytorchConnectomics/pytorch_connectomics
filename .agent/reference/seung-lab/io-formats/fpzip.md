# fpzip

**GitHub:** https://github.com/seung-lab/fpzip
**Language:** C++ | **Stars:** 36

Python/Cython bindings for the fpzip floating-point compression algorithm (v1.3.0). Supports lossless and lossy compression of up to 4D float/double arrays.

## Key Features
- Lossless and lossy floating-point compression
- Up to 4D numpy array support (float32, float64)
- C and Fortran memory order support
- Configurable precision for lossy mode

## API
```python
import fpzip
import numpy as np

data = np.array(..., dtype=np.float32)
compressed = fpzip.compress(data, precision=0, order='C')  # lossless
data_again = fpzip.decompress(compressed, order='C')
```

## Relevance to Connectomics
Compresses floating-point affinity maps and probability volumes generated by EM segmentation networks, reducing storage for large-scale connectomics datasets.
