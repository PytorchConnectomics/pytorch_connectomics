# EMIRT.jl

**GitHub:** https://github.com/seung-lab/EMIRT.jl
**Language:** Julia | **Stars:** 2

Electron Microscopy Image Reconstruction Toolbox for Julia. Provides image processing and analysis utilities for EM segmentation and neuron reconstruction workflows.

## Installation
```julia
Pkg.add("EMIRT")
```

## Relevance to Connectomics
Julia utility library for EM image processing, the Julia counterpart of the Python emirt package.
