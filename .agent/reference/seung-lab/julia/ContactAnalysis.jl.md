# ContactAnalysis.jl

**GitHub:** https://github.com/seung-lab/ContactAnalysis.jl
**Language:** Julia | **Stars:** 0

Simple weighted count of edges shared by neighboring voxels between segment pairs. Quantifies contact surface area between neuron segments.

## Key Features
- Count shared voxel faces between segment pairs
- Weighted edge counting for contact analysis

## Relevance to Connectomics
Used to quantify physical contact between reconstructed neurons, a prerequisite for synapse detection and connectivity analysis.
