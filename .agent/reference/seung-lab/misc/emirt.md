# emirt

**GitHub:** https://github.com/seung-lab/emirt
**Language:** Python | **Stars:** 0

EM Image Reconstruction Toolbox -- a Python library for 3D image processing and analysis built for EM image segmentation and neuron reconstruction in Seung Lab.

## Key Features
- 3D image processing utilities for EM data
- Segmentation and neuron reconstruction helpers
- General-purpose connectomics image analysis

## Installation
```bash
pip install emirt
```

## Relevance to Connectomics
Python utility library providing image processing primitives used in EM segmentation pipelines.
