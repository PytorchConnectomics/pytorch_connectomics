# MicronsBinder

**GitHub:** https://github.com/seung-lab/MicronsBinder
**Language:** Jupyter Notebook | **Stars:** 3

Collection of Jupyter notebooks demonstrating how to use data from microns-explorer.org, the MICrONS cubic millimeter EM + functional imaging dataset. Designed for mybinder.org.

## Key Features
- CAVE (Connectome Annotation Versioning Engine) setup tutorials
- Synapse and annotation querying examples
- Mesh access and visualization (CloudVolume + MeshParty)
- Phase 1 and Phase 3 dataset notebooks
- Motif analysis, functional analysis, mitochondria analysis vignettes

## Relevance to Connectomics
Provides example workflows for analyzing the MICrONS dataset, one of the largest EM connectomics volumes with correlated functional data.
